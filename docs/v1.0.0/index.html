    Welcome to gallerynpy’s documentation!

    gallerynpyx is a set of Ren’Py scripts that allows you to display a gallery screen in your visual novel. The gallery screen displays all images, animations and videos that have been inserted through the methods provided by the gallerynpyx handler.

    gallerynpyx is a new improved, modularized and optimized version of gallerynpy (v2.0). It improves usability and customization, adds events and aims to allow the integration of gallery extensions without much difficulty.
