    helpers

    cast(obj, caster: Callable[[Any, Ellipsis], _T], default: _T = None, allowed_exceptions: tuple[type, Ellipsis] = None, **kwargs) → _T

    Attempts to cast an object to a specified type.

    Notes:

    If the allowed_exceptions is not specified, all exceptions will be caught.

    Parameters:

    • obj – The object to cast.

    • caster – The casting function or type to use.

    • default – Value to return if casting fails.

    • allowed_exceptions – Tuple of exceptions to catch during casting

    • kwargs – Additional keyword arguments passed to the caster

    Returns:

    The cast object or default value if casting fails.
