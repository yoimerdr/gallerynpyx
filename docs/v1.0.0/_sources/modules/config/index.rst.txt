config
======

This module contains the submodules with classes corresponding to the different available configurations
for the management and customization of the parameters in the gallery.

In addition, it provides short imports that allow direct access to obtain the configuration instances.

* :func:`get_resources <gallerynpyx.config.resources.ResourcesConfig>`
* :func:`get_screens <gallerynpyx.config.screens.ScreensConfig>`
* :func:`get_styles <gallerynpyx.config.styles.StylesConfig>`
