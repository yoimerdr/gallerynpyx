    base

    class BaseHandler

    Bases: object

    Core gallery handler with paging, routing and slide navigation support.

    Creates the root slider and initializes the default page distribution.

    change(name: AnyStr | None, *names: AnyStr) → None

    Changes the active route, slide or slider.

    Parameters:

    • name – First route segment, or None to return to the root.

    • names – Additional route segments.

    change_distribution(rows: int | None = None, cols: int | None = None) → None

    Updates the number of rows and columns used by each page.

    Parameters:

    • rows – Optional new number of rows.

    • cols – Optional new number of columns.

    next() → bool

    Advances the page in the active slide.

    previous() → bool

    Goes back one page in the active slide.

    put(names: AnyStr | Iterable[AnyStr] | SlideBase, resource: Resource | AnyStr | Movie | ATLTransform | Image | Displayable, song: AnyStr | None = None, condition: Callable[[Item], bool] | None = None, tooltip: AnyStr | None = None, locked_tooltip: AnyStr | None = None, label: AnyStr | None = None, thumbnail: Resource | AnyStr | Movie | ATLTransform | Image | Displayable | None = None) → bool

    Creates an item and adds it to the given route.

    Parameters:

    • names – Route, route parts or slide container target.

    • resource – Resource source assigned to the new item.

    • song – Optional song path to play while the item is shown.

    • condition – Optional callable used to determine whether the item is unlocked.

    • tooltip – Optional tooltip shown when the item is available.

    • locked_tooltip – Optional tooltip shown when the item is locked.

    • label – Optional label used when creating the target slide.

    • thumbnail – Optional custom thumbnail source.

    put_item(names: AnyStr | Iterable[AnyStr] | SlideBase, item: Item, label: AnyStr | None = None, thumbnail: AnyStr | Resource | Displayable | ATLTransform | Image | Movie | None = None) → bool

    Adds an existing item to the given route.

    Parameters:

    • names – Route, route parts or slide container target.

    • item – Item to add.

    • label – Optional label used when creating the target slide.

    • thumbnail – Optional custom thumbnail source.
