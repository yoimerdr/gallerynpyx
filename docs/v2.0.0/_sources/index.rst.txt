gallerynpyx Documentation
=========================

gallerynpyx is a modular gallery library for Ren'Py games. It can display images,
animations and videos registered in a gallery handler, while keeping the screen layer,
styles and resources configurable.

Version ``2.0.0`` introduces gallery-scoped handlers and configurations. The shared
gallery remains available through the default accessors, and you can now create named
local galleries with their own handler state, styles, screens and resource settings.
The built-in screens and actions were updated to work with whichever handler is passed
through ``gx_handler``.

Highlights in ``2.0.0``:

* Shared and named local galleries.
* Shared and local configuration managers for resources, screens and styles.
* Handler-aware built-in screens and actions.
* Better extraction of ATL animation statements.
* Thumbnail prediction support for slide and page actions.
