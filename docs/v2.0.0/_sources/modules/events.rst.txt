events
------

This module provides functions for subscribing to the gallery events emitted by handlers.
In ``2.0.0``, slide and page events also receive the active handler instance, which makes
it possible to react differently depending on the gallery scope that triggered the event.
The handler manager also emits a flush event when cached galleries are cleared.
