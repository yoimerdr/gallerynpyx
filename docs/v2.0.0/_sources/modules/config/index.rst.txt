config
======

This package contains the configuration types used to customize gallery resources,
screens and styles.

In ``2.0.0``, the same accessors can resolve either the shared configuration objects or
named local ones. The helper functions also accept existing configuration instances, so
custom code can work with both flows transparently.

* :func:`get_resources <gallerynpyx.config.get_resources>`
* :func:`get_screens <gallerynpyx.config.get_screens>`
* :func:`get_styles <gallerynpyx.config.get_styles>`
* :func:`coerce_resources <gallerynpyx.config.coerce_resources>`
* :func:`coerce_screens <gallerynpyx.config.coerce_screens>`
* :func:`coerce_styles <gallerynpyx.config.coerce_styles>`
